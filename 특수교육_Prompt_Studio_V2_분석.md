## 1. 서비스 개요

### 1.1 서비스 정체성
**특수교육 Prompt Studio V2**는 특수교사가 ChatGPT·Claude·Gemini 등 생성형 AI에 입력할 **맞춤형 프롬프트를 칩(Chip) UI로 빠르게 조립**하도록 돕는 웹 도구이다. 학생 특성·교육과정·교수법·접근성 옵션을 칩으로 선택하면 자동으로 구조화된 프롬프트 문자열이 생성되고, 클립보드에 복사되어 AI에 붙여넣기만 하면 된다.

### 1.2 핵심 가치
- **2022 개정 기본교육과정 연계** — 기본·공통·선택 중심 교육과정 및 IEP 기반 맞춤
- **수업 자료 생성 특화** — 웹 앱·문서·시각 지원 자료 3개 결과물 계열
- **근거 기반 실제(EBP) 내장** — ABA, DTT, UDL, PBS, PECS, Social Story 등
- **접근성 우선 설계** — TTS, 큰 터치, 고대비, AAC 연계 등

### 1.3 권장 사용 환경
- **Claude Artifacts** (최우선 권장) — HTML 단일 파일 즉시 실행
- **ChatGPT Canvas / Gemini Canvas** — 보조 지원

## 3. 프롬프트 빌더 페이지

### 3.1 구조 개요
좌측 카테고리 패널(0~7번 8개)과 우측 입력창(자동 입력되는 텍스트 칸)의 2단 구성. 칩을 클릭하면 우측에 해당 옵션이 자동 누적된다.

### 3.2 카테고리 8종 요약

| 번호 | 카테고리 | 필수 여부 |
|------|----------|-----------|
| 0 | 학생 프로파일 (Student Profile) | 권장 |
| 1 | 결과물 형태 | **★ 필수** |
| 2 | 교육과정 & 수업 내용 | 핵심 추가 |
| 3 | 수업 수정 & 설계 | ★ |
| 4 | IEP 목표 & 평가 계획 | 신규 |
| 5 | 특수교육 교수법 (EBP) | — |
| 6 | 접근성 & 보조공학 | — |
| 7 | 디자인 & 형식 옵션 | — |

### 3.3 단축키 지원
- **Ctrl+Enter** — 프롬프트 생성 및 복사 (전 영역 공통)

### 3.4 안내 배너

**상단 안내 #1**
> 👈 **옵션 선택** 탭에서 칩을 누르면 여기에 자동 입력돼요. 아래 칸에 직접 입력도 가능해요!

**상단 안내 #2**
> 💡 **사용법:** 왼쪽 칩 클릭 → 자동 입력. **1번 결과물 형태**만 선택해도 생성 가능! 수업 자료를 만들려면 2번(교육과정·수업내용)과 3번(수업 수정)을 꼭 채워주세요.

## 8. 옵션 선택 탭 — 4. IEP 목표 & 평가 계획 (신규)

### 8.1 📋 IEP 목표 형식 (4종)

| 형식 | 설명 |
|------|------|
| 📐 조건-행동-기준 | 3요소 구조(예: "교사의 언어촉진이 주어지면(조건), 학생은 1~5 숫자를(행동) 3번 중 2번 정확히 말한다(기준)") |
| 🎯 SMART | Specific, Measurable, Achievable, Relevant, Time-bound |
| 📆 연간·단기 분리 | 장기목표(Annual Goal) + 단기목표(Short-term Objective) |

## 15. 가이드 페이지 — 입력 칸별 작성법

### 15.1 [0] 학생 프로파일 — AI의 언어·수준 기준점

장애 유형과 기능 수준 선택 시 AI가 **어휘 수준·문장 길이·시각 지원 방식을 자동 조정**. "같은 '수세기' 수업도 지적장애 입문 수준 vs 자폐스펙트럼 중급은 완전히 다른 자료가 나와요." 기능 수준을 선택하지 않으면 일반 수준으로 생성.

### 15.2 [1] 결과물 형태 ★필수 — 가장 중요한 한 칸

- 나머지를 비워도 이것만 선택하면 AI가 자동 완성
- **웹 앱 계열**은 HTML 코드로, **문서 계열**은 마크다운으로 자동 전환
- ⚠️ 한 가지만 선택 — 두 개 이상 선택 시 결과 품질 저하

### 15.3 [2-A] 교육과정 유형·학년군·교과 — 수준 결정의 핵심

- **기본 교육과정** vs **공통 교육과정**은 교과명·성취기준이 완전히 다름
- 초등 1~2학년 기본 교육과정은 **바른생활·슬기로운생활·즐거운생활(통합교과)**
- 초등 3학년부터 분과 교과 시작
- 학년군 미선택 시 AI가 일반 교과서 수준으로 자료 생성

### 15.4 [2-B] 수업 주제 & 학습 내용 ★★ — 자세할수록 맞춤형

**입력 권장 내용**
- 단원명
- 학습 목표
- 핵심 어휘
- 학생 현재 수준
- 교과서 원문 붙여넣기 가능
- **학생 관심사**(공룡·기차 등) 포함 시 동기 유발 자료 생성

### 15.5 [3] 수업 수정 & 설계 구조 — 특수교육 자료의 차별점

**핵심 수정 유형**
- **축소** — 내용·분량 줄이기
- **대체** — 기능적 목표로 바꾸기
- **간소화** — 같은 주제, 더 쉽게

추가 선택 항목: 수업 인원, 협력교수 유형, 도입-전개-정리 전략 → 지도안 품질 상승.

### 15.6 [4] IEP & 평가 계획 — 문서형 자료에 특히 중요

- 수업 지도안·IEP 목표 초안 생성 시 평가 체크리스트·데이터 수집 시트 자동 첨부
- 일반화 계획(가정·지역사회 확장) 선택 시 보호자 안내 문구 자동 생성

## 18. 가이드 페이지 — AI 활용 팁 및 주의사항

### 18.1 💡 AI 200% 활용 팁 5가지

**🔁 팁 1 — 반복 대화로 완성도 끌어올리기**
"버튼 더 크게", "문항 10개로 늘려줘", "더 쉬운 말로 바꿔줘" 식으로 3~5회 주고받기.

**📋 팁 2 — 교과서·활동지 그대로 붙여넣기**
2-B 칸에 교과서 본문·기존 활동지·참고 자료 그대로 붙여넣기 가능. 타이핑 불필요.

**💬 팁 3 — 학생 관심사 활용**
"공룡을 좋아함", "버스 번호에 관심이 많음" 등 관심사 추가 시 동기 유발 자료 생성.

**🖥️ 팁 4 — Claude Artifacts 최적화**
- 웹 앱 계열은 Claude **Artifacts 모드(캔버스)**에서 즉시 실행
- ChatGPT·Gemini는 Canvas 모드 활성화 필요
- "Artifacts로 만들어줘" 재요청 권장

**💾 팁 5 — HTML 오프라인 저장**
생성된 HTML 코드를 메모장에 붙여 `파일명.html`로 저장 → 오프라인에서 학교 PC·태블릿·USB에서 사용 가능.

### 18.2 반드시 알아두세요 (4대 주의사항)

**⚠️ AI는 보조 도구**
생성된 자료는 특수교사가 반드시 검토·수정 후 사용. **IEP 목표·행동지원계획은 전문가 판단 필수**.

**🔒 학생 개인정보 절대 입력 금지**
이름·생년월일·학번 금지. "A학생" 또는 "대상 학생"으로 표기.

**📋 교육과정 성취기준 대조**
AI가 제시하는 성취기준 코드는 오류 가능. **국가교육과정정보센터(ncic.re.kr)** 에서 반드시 확인.

**🎨 저작권**
AI 생성 자료라도 **상업적 배포·대외 공개 시 주의**. 교실 내 사용은 자유.
